Therefore the optimization should be transformed from the multiplication to addition, and the solution is by using logarithm:

Why using logarithm?

  • Log of the multiplication is the addition.
  • The log is a monotonically increasing function, the function reaches the maximum value if the logarithm of the function reaches the maximum and vice versa.
