CNN Architecture

LeNet-5

Created by Yann Lecun, 1998.

Recognize handwriting.

Weight size: 60K

AlexNet

Architecture like LeNet-5

Weight size: 60M (> 1000 times LeNet-5)

VGG-16

Created in 2014.

Simpler than AlexNet, LeNet.

However, many layers → many weights, deeper

  • Filter 3x3.
  • Stack of 3:

    Increase non-linearity (3 activations).

    Fewer parameters (3 x 3) x 3 = 27 < (7 x 7) x 1 = 49

Weight sizes: 138M

Feed 1 image → VGG-16 = 100 MB / image.

Backpropagation → 1 image = 200 MB .

⇒ 16 images = 200 x 16 = 3 GB.

⇒ Divide into batch size.

Inception (GoogLe Net)

2014, Google → For ImageNet 2014.

  • No FC layers.
  • Used for medical purposes.

Weight sizes:

  • Inception v1: 5M
  • Inception v3: 23M

1x1 Conv: 28 x 28 x 256 x 128 x 1 x 1 = 25 M ops.

3x3 Conv: 28 x 28 x 256 x 192 x 3 x 3 = 346 M ops.

5x5 Conv: 28 x 28 x 256 x 96 x 5 x 5 = 481 M ops.

⇒ 854 M ops on floating type ⇒ 854 x 4 = 3416 M ops.

⇒ Huge parameters, large computation complexity. However, it sums up information from all kernels.

Bottleneck

1x1 Conv (128 filters): 28 x 28 x 256 x 128 x 1 x 1 = 25 M ops.

3x3 Conv: (28 x 28 x 256 x 64 x 1 x 1) + (28 x 28 x 64 x 192 x 3 x 3) = 99 M ops.

5x5 Conv: (28 x 28 x 256 x 64 x 1 x 1) + (28 x 28 x 64 x 96 x 5 x 5) = 133 M ops.

1x1 Conv (64 filters): 28 x 28 x 256 x 64 x 1 x 1 = 12 M ops.

GoogleNet

GoogleNet has 3 outputs: Avoid Gradient Vanishing (if loss Output 1 tends to go to 0, loss Output 2 and Output 3 will increase the total loss).
